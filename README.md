# Viet Sport WP Theme

Theme child tùy biến từ SWELL để phục vụ dự án viet-sport.com  
Bao gồm CPT cho CLB thể thao, tính năng đặt sân (booking) mời bạn chơi cùng.

## Cấu trúc thư mục

- includes/
- booking/
- functions.php
- ...

## Hướng dẫn chạy local
1. Clone repo
2. Copy vào wp-content/themes
3. Active trong admin
